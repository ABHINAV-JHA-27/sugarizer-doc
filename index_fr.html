Un goût de Sugar sur n'importe quelle machine

Avec Sugarizer, vous pouvez découvrir la plate-forme pédagogique Sugar sur n'importe quelle machine

Développée pour le projet One Laptop per Child, Sugar est utilisée par presque 3 millions d'enfants partout dans le monde. Avec Sugarizer, Sugar est maintenant sur votre machine.

Exécuter maintenant

Sugarizer est distribué de 3 manières:

Client léger

Le Client léger permet d'exécuter Sugarizer depuis un navigateur. Il peut fonctionner sur n'importe quelle machine avec une version récente de Chrome/Chromium, il a aussi été testé avec succès sur Firefox, Safari et IE. Le Client léger Sugarizer ne nécessite aucune installation sur la machine mais doit disposer d'un accès permanent à un Serveur Sugarizer.

Client

Sugarizer est installé localement sur la machine, il ne nécessite donc aucun accès Serveur - il peut fonctionner de manière autonome. Sugarizer Client est disponible pour PC/MacOS, Android, iOS et Chrome Web App.

Serveur

Le Serveur Sugarizer est le coeur des fonctions réseau de Sugarizer. Cela signifie exposer le Client léger pour permettre le déploiement de Sugarizer sur une serveur local, par exemple un serveur d'école. Le Serveur Sugarizer permet aussi de fournir à travers le réseau des fonctions de collaboration aux Clients et Clients légers.
