Un toque de Sugar en cualquier dispositivo

Con Sugarizer, puedes descubrir la Plataforma de Aprendizaje Sugar en cualquier dispositivo

Usada diariamente por casi 3 millones de niños alrededor del mundo, Sugar es una plataforma de aprendizaje líder desarrollada para el proyecto One Laptop per Child. Con Sugarizer, está ahora en tu dispositivo.

Pruébalo Descarga Código

Sugarizer es distribuido en forma de 3 componentes:

Cliente ligero

Funciona en cualquier dispositivo con una versión reciente de Chrome/Chromium, y también ha sido probada con éxito en Firefox, Safari e Internet Explorer. El cliente ligero de Sugarizer no necesita ninguna instalación en el dispositivo pero requiere un permanente acceso a un servidor de Sugarizer.

Cliente

Sugarizer instalado localmente en el dispositivo por lo que no requiere ningún acceso al servidor - funciona independientemente. El cliente de Sugarizer está disponible para PC/Mac, Android, iOS y como aplicación web para Chrome.

Servidor

El servidor de Sugarizer es el back-end para las funciones de red de Sugarizer. Expone localmente el cliente ligero permitiendo el despliegue de Sugarizer en un servidor local, por ejemplo, en un servidor escolar. El servidor de Sugarizer puede también ser usado para proveer funciones de colaboración para el cliente y el cliente ligero en la red.
