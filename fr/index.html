Un goût de Sugar sur n'importe quelle machine

La plate-forme pédagogique Sugar est la première plate-forme pédagogique au monde. Développée pour le projet One Laptop per Child, elle est utilisée par presque 3 millions d'enfants partout dans le monde. Avec Sugarizer, vous pouvez découvrir la plate-forme pédagogique Sugar sur n'importe quelle machine: du minuscule RaspberryPI au gros PC/Mac en passant par la tablet Android/iOS. Profitez de l'expérience et aidez-nous à faire en sorte que Sugar atteigne tous les enfants sur toutes les machines dans tous les pays.

Sugarizer est disponible de 3 manières:

  • Client léger: un accès distant à Sugarizer via un navigateur,
  • Client: Sugarizer s'exécutant localement,
  • Serveur: pour exécuter les fonctionnalités distante de Sugarizer.

Client léger

Le Client léger permet d'exécuter Sugarizer depuis un navigateur. Il peut fonctionner sur n'importe quelle machine avec une version récente de Chrome/Chromium, il a aussi été testé avec succès sur Firefox, Safari et IE. Le Client léger Sugarizer ne nécessite aucune installation sur la machine mais doit disposer d'un accès permanent à un Serveur Sugarizer.

Exécuter maintenant !

Les fonctionnalités du Client léger Sugarizer sont notamment:

  • Les 3 vues du bureau Sugar (Radial, Liste et Journal),
  • Stockage local du datastore - limité par le navigateur à 5Mo (voir ici),
  • Exécution d'activités Sugar Web stockées à distance,
  • Sauvegarde et partage des contenus locaux sur le serveur (Pas encore implementé),
  • Collaboration (Pas encore implementé)

Vous pouvez exécuter Sugarizer ici mais chaque Serveur Sugarizer peut également fournir un accès Client léger Sugarizer.

Client

Le Client Sugarizer est installé localement sur la machine, il ne nécessite donc aucun accès Serveur - il peut fonctionner de manière autonome. Des accès Serveur peuvent néanmoins être nécessaires pour utiliser les fonctionnalités réseau. Il peut fonctionner sur n'importe quelle machine avec une version récente de Chrome/Chromium, il a aussi été testé avec succès sur Firefox, Safari et IE. Le Client Sugarizer est disponible sur PC et sur Android.

Pour installer Sugarizer sur un PC (GNU Linux/Mac OS/Windows), télécharger le fichier org.olpc-france.sugarizer.zip, décompresser son contenu puis lancer Chrome en utilisant la ligne de commande:

	chrome --allow-file-access-from-files

Ouvrir alors le fichier "index.html" dans le navigateur. Noter que l'option "--allow-file-access-from-files" est nécessaire pour autoriser l'accès aux fichiers locaux. Ne pas oublier de fermer toutes les instances de Chrome avant d'utiliser cette commande. Pour des commandes équivalentes pour les autres navigateurs, voir ici. N'hésiter pas à créer un raccourci sur cette commande pour les lancements suivants.

Télécharger pour PC

Pour installer Sugarizer sur Android, commencer par "autoriser l'installation depuis des sources inconnues" dans les paramètres Android. Cela permet l'installation d'applications ne provenant pas de Google Play (voir ici pour apprendre comment). Puis télécharger le fichier org.olpc-france.sugarizer.apk depuis le navigateur de votre Android et accepter les conditions d'installation.

Les fonctionnalités du Client Sugarizer sont notamment:

  • Les 3 vues du bureau Sugar (Radial, Liste et Journal),
  • Stockage local du datastore - limité par le navigateur à 5Mo (voir ici),
  • Exécution d'activités Sugar Web stockées localement,
  • Possibilité de télécharger localement de nouvelles activités en provenance du Serveur (Pas encore implementé),
  • Sauvegarde et partage des contenus locaux sur le serveur (Pas encore implementé),
  • Collaboration (Pas encore implementé)

Serveur

Le Serveur Sugarizer est le coeur des fonctions réseau de Sugarizer. Actuellement, cela signifie simplement exposer le Client léger pour permettre le déploiement de Sugarizer sur une serveur local, par exemple un serveur d'école. Dans un futur proche, le Serveur Sugarizer permettra aussi de fournir à travers le réseau des fonctions de collaboration aux Clients et Clients légers. Le Serveur Sugarizer peut être déployé sur n'importe quel ordinateur disposant d'un Serveur Web et de Node.js.

Télécharger le Serveur

Les fonctionnalités du Serveur Sugarizer sont notamment:

  • Accès Client léger Sugarizer,
  • Gestion de la collaboration entre les Clients/Clients légers sur le réseau (Pas encore implementé),
  • "Store" d'activités pour les Clients (Pas encore implementé),
  • Stockage des sauvegardes et partages pour les Clients et Clients légers (Pas encore implementé).

Une instance du Serveur Sugarizer (uniquement l'accès Client léger) est utilisable ici. Pour exécuter votre propre Serveur Sugarizer, décompresser simplement le contenu du Client Sugarizer dans un répertoire de votre Serveur Web. Node.js n'est pas nécessaire pour la version actuelle.
